# Time complexities

Table of time complexities for algorithms in this repo

## Sorting

| algorithm | complexity   |
| --------- | ------------ |
| bubble    | O(n^2)       |
| selection | O(n^2)       |
| insertion | O(n^2)       |
| merge     | O(n\*log(n)) |

## Searching

| algorithm | complexity |
| --------- | ---------- |
| linear    | O(n^2)     |
| binary    | O(log n)   |
